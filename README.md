## Deploy no Vercel

Este projeto está configurado para ser implantado no Vercel automaticamente através do GitHub ou via CLI.

### Configurações sugeridas:
- **Framework Preset**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Install Command**: `npm install`

O arquivo `vercel.json` foi adicionado para garantir que as rotas do React (SPA) funcionem corretamente após o deploy.
